# Datawhale - Happy LLM

## 项目简介

Happy-LLM 是一个从零开始的大语言模型原理与实践教程。这是一个系统性的LLM学习教程，将从NLP的基本研究方法出发，根据LLM的思路及原理逐层深入，依次为读者剖析LLM的架构基础和训练过程。

## 学习指南

### 适用人群
- 大学生
- 研究人员
- LLM爱好者
- 想要深入理解LLM原理的开发者

### 学习前准备
- 一定的编程经验（Python）
- 深度学习相关知识
- NLP领域基本概念和术语
- 基础的数学知识

### 学习路径

#### 基础知识部分（第1-4章）
- **第1章**: NLP基本任务和发展（为非NLP研究者提供参考）
- **第2章**: LLM基本架构——Transformer（核心理论基础）
- **第3章**: 经典PLM架构（三种架构对比）
- **第4章**: LLM特点和训练过程（正式进入LLM部分）

#### 实战应用部分（第5-7章）
- **第5章**: 基于PyTorch亲手搭建LLM
- **第6章**: 基于Transformers框架实现LLM训练
- **第7章**: LLM应用（评测、RAG、Agent）

### 学习建议

1. **理论结合实践**: 复现教程中的各种代码
2. **多投入实战**: 参加LLM相关的项目和比赛
3. **社区互动**: 在issue区提问和讨论
4. **关注更新**: 关注Datawhale社区的最新动态
5. **贡献代码**: 欢迎提交PR改进内容
