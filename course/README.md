# LLM 学习课程编目

欢迎来到大语言模型（LLM）学习课程编目！本编目收集了国内外最优质的LLM学习资源，涵盖从入门到高级的完整学习路径。

## 📚 编目介绍

本编目精心筛选了7个优秀的LLM学习课程，包括中文课程和英文课程，适合不同基础和需求的学习者。每个课程都有自己的特色和优势，可以根据自己的兴趣和学习目标选择合适的课程。

## 🗂️ 课程目录

### 🇨🇳 中文课程

#### 1. 李宏毅大模型教程
- **链接**: [B站视频](https://www.bilibili.com/video/BV1JWy3BkEU4/)
- **类型**: 视频教程（入门到进阶）
- **特点**: 2026年公认最好的LLM大模型教程，附带课件和代码
- **适合人群**: 想要系统学习大语言模型原理的初学者
- **学习时间**: 数十小时视频内容

#### 2. Datawhale - LLM Cookbook
- **链接**: [GitHub仓库](https://github.com/datawhalechina/llm-cookbook)
- **类型**: 开源教程（实践导向）
- **特点**: 面向开发者的大模型手册，基于吴恩达课程中文版
- **适合人群**: 具备基础Python能力的开发者
- **学习时间**: 2-3周（建议按必修→选修顺序学习）

#### 3. Datawhale - Happy LLM
- **链接**: [GitHub仓库](https://github.com/datawhalechina/happy-llm)
- **类型**: 系统教程（原理深入）
- **特点**: 从零开始的大语言模型原理与实践，亲手实现LLaMA2
- **适合人群**: 想要深入理解LLM原理的开发者
- **学习时间**: 4-6周（理论基础+代码实现）

### 🇺🇸 英文课程

#### 4. Coursera - Generative AI with Large Language Models
- **链接**: [课程地址](https://www.coursera.org/learn/generative-ai-with-llms)
- **类型**: 专业认证课程
- **特点**: DeepLearning.AI + AWS联合出品，3个完整模块，包含项目
- **适合人群**: 有Python背景的LLM学习者
- **学习时间**: 3-4周

#### 5. Salesforce - LLM Course
- **链接**: [课程地址](https://www.salesforce.com/agentforce/llm-course/)
- **类型**: 微学习课程
- **特点**: Salesforce免费LLM培训，实践导向，Agentblazer认证
- **适合人群**: AI初学者，业务人员
- **学习时间**: 5-10小时

#### 6. Google Skills - Introduction to Large Language Models
- **链接**: [课程地址](https://www.skills.google/course_templates/539)
- **类型**: 快速入门课程
- **特点**: Google出品，10分钟微学习，中文支持
- **适合人群**: 零基础学习者，快速了解LLM
- **学习时间**: 10分钟

#### 7. ActiveLoop - Training & Fine-Tuning LLMs for Production
- **链接**: [课程地址](https://learn.activeloop.ai/courses/llms)
- **类型**: 高级实战课程
- **特点**: Activeloop+Towards AI+Intel合作，67节课，10个项目
- **适合人群**: 高级学习者，想要生产级部署
- **学习时间**: 25-40+小时

## 📋 选择指南

### 按学习阶段选择

#### 🌱 初学者
1. **快速入门**: Google Skills - Introduction to Large Language Models
2. **中文入门**: 李宏毅大模型教程
3. **理论基础**: Datawhale - Happy LLM（前4章）

#### 🚀 进阶学习
1. **实践项目**: Datawhale - LLM Cookbook（必修课程）
2. **系统学习**: Coursera - Generative AI with Large Language Models
3. **业务应用**: Salesforce - LLM Course

#### 💼 高级/专业
1. **深入原理**: Datawhale - Happy LLM（后3章）
2. **生产部署**: ActiveLoop - Training & Fine-Tuning LLMs for Production
3. **专题优化**: Datawhale - LLM Cookbook（选修课程）

### 按目标选择

#### 理论研究
- Happy-LLM → Coursera GenAI → LLM Cookbook选修

#### 实际开发
- LLM Cookbook → Happy-LLM → ActiveLoop

#### 业务应用
- 李宏毅教程 → Salesforce LLM → Coursera GenAI

#### 快速上手
- Google Skills → Salesforce LLM → LLM Cookbook必修

## 🎯 学习建议

### 1. 循序渐进
- 先打好基础，再深入高级内容
- 理论学习与动手实践相结合
- 不要急于求成，持续学习

### 2. 动手实践
- 每个课程都有配套代码，务必亲手运行
- 尝试修改代码，理解原理
- 做自己的项目，巩固所学

### 3. 持续更新
- AI领域发展迅速，关注最新进展
- 加入社区，与其他学习者交流
- 定期复习，保持知识更新

### 4. 资源整合
- 可以同时学习多个课程，取长补短
- 参考不同课程的观点和方法
- 建立自己的知识体系
